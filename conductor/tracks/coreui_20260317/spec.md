# Specification: Implement core UI and SteamCMD server installation

## 1. Overview
The goal of this track is to build the initial scaffolding of the Icarus Sentinel application, including the main UI structure using CustomTkinter and the foundational SteamCMD integration to download and install the dedicated server.

## 2. Requirements Addressed
* **F1:** SteamCMD Integration
* **F5:** Basic Configuration UI (partial)

## 3. Scope
* Set up a minimal Python CustomTkinter UI with basic styling based on the Native Windows 11 guideline.
* Implement the background downloading and execution of SteamCMD.
* Provide user feedback during the download phase.

## 4. Out of Scope
* RAM/CPU tracking (will be a separate track)
* Backups and `.zip` generation
