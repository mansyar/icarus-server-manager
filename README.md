# Icarus Sentinel

A modern, portable server manager for Icarus, designed for simplicity and reliability.

## Features
-   **One-Click Installation:** Automated SteamCMD integration for easy server setup.
-   **Resource Monitoring:** Real-time CPU and RAM tracking with smart threshold alerts.
-   **Smart Idle Restart:** Automatically restarts your server during idle periods to maintain performance.
-   **Automated Backups:** Configurable backup intervals and retention policies for your world data.
-   **Save Synchronization:** Bidirectional sync between local and dedicated server saves.
-   **Portability:** Single executable distribution - no Python installation required.


## Installation
1.  **Download:** Download the latest `IcarusSentinel.zip` from the [Releases](https://github.com/your-username/icarus-server-manager/releases) page.
2.  **Extract:** Extract the contents of the `.zip` to a folder of your choice.
3.  **Run:** Launch `IcarusSentinel.exe`.

## Quick Start
1.  **Select Path:** In the "Server" tab, select or enter the directory where you want to install the Icarus Dedicated Server.
2.  **Install:** Click the **Install/Update Server** button to download the server files (this may take some time).
3.  **Configure:** Switch to the "Configuration" tab to set your server name, password, and port.
4.  **Start:** Click **Start Server** on the "Server" tab.
5.  **Enjoy:** Your server is now running! Connect via the Icarus game client.

## Troubleshooting
-   **Windows SmartScreen:** Because the executable is not code-signed, Windows may show a warning. Click "More info" and "Run anyway".
-   **SteamCMD Errors:** Ensure you have a stable internet connection. If the installation fails, try clicking "Install/Update Server" again.
-   **Port Forwarding:** Ensure the configured ports (default 17777 and 27015) are forwarded in your router/firewall if you want to allow players from outside your network.
-   **RAM Usage:** If you experience frequent restarts, check the "RAM Threshold" setting on the Server tab. Icarus can be memory-intensive.
